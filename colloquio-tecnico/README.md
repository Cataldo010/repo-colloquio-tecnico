## Obiettivo
Implementare delle Rest API per la creazione e visualizzazione del modello user, In particolare dovrai creare i seguenti endpoint con i rispettivi metodi per la scrittura/lettura sul db:
- createUser
- getUserById

## Servizi AWS
- Lambda Function
- DynamoDB
- API Gateway

## Framework

- Serverless
